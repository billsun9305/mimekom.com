FAQ

Frequently Asked Questions

Installation

How to install the Slim SIM?

1 Tear 2 Place 3 Stick 4 Press. For details, please refer to youtube video demonstration here

If the Chip manager prompts that the Slim SIM can’t be detected.

Please ensure if you have installed the Slim SIM according to the steps on the package.

What is the original PIN code?

For the first time usage, please enter the PIN code as 111111.

Why do you need to disable the battery performance optimized function?

As the safety detection is continuously working in the background, please disable the "unused applications into the sleep mode", "automatically disable unused applications" or "battery performance optimized" in order to provide the comprehensive MIMEKOM service.

Where is able to disable the data saving mode in different Android versions?

For Android 7.0, please click manage applications from the application and authority to click the data usage of MIMEKOM, and enable the data usage without restrictions.


For Android 8.0, please click manage applications from the application and permissions to click the data usage of MIMEKOM, and enable the data usage without restrictions.
For Android 9.0, please click manage applications from the application and permissions to click the data usage of MIMEKOM, and enable the data usage without restrictions.

Registration

If you lost or forgot your password?

As MIMEKOM is not record user password, so we won’t be able to offer the password reset service and please ensure that you have record the password while registration.

Can the default number of the registered account be changed?

Yes, it can be changed once.

Usage Questions

Which users have group management?

Only the group owner has team management.

How do you send an Evaporated message?

In the chat page, click on the "lightning bolt" on the left side of the dialog box to enter the content, and click send.

How soon will the Evaporated message be destroyed?

The timing starts when the user opens the chat page to view the Evaporated message. The default is 9 seconds (the user can set the destroy timer: 9 seconds, 15 seconds, 30 seconds, 60 seconds) and mark this Evaporated message be destroyed.

What is the feature for anti-screenshot?

Prevent malicious leakage of data by intercepting the screen, in order to protect the security of MIMEKOM.

What is the upper limit of group and the number of groups?

The maximum number of members in each group is 500, and each user can create or join up to 200 groups.

If you want to add a large number of friends?

The Business Group Contacts can support the selection of 2000 people/times.

If the recording prevention function is disabled, where can you find the recording files?

Please go to the “chat & call” from “More”, and click the “call recording”, you can find the recording file.

How do you transform messages as ciphertext?

Please click “More”-->“Account and Security”-->Plaintext Mode, please disable the Plaintext display. After returning to the message page, you can only see the message composed of garbled characters and can’t send any new messages. To return the Plaintext, please enter the login password for authentication, and all the messages will be decoded.

Why can't log in MIMEKOM?

If the phone has been rebooted, please ensure the Chip Manager App is running and able to detect the Slim SIM.

If the MIMKOM App crashes or becomes unstable?

For Android users, please reboot and try it again. If you still can’t use it after rebooting, please remove the Chip Manager APP and the MIMEKOM APP and reinstall them.
For iOS users, please remove the MIMEKOM APP and reinstall it. If it is not available, please contact MIMEKOM customer service and inform the phone model and the card id on the Slim SIM card body.

If the MIMEKOM app is not workable?

For Android users, please make sure to stick Slim SIM first and install the Chip Manager.
For iOS users, please make sure to stick Slim SIM first and update the iOS version to the latest version

If the Slim SIM is damaged?

Please inform MIMEKOM customer service the card id on the card body. If there is obviously damage on Slim SIM, we will provide a free replacement for the Slim SIM.
If there is no damage on Slim SIM, it is recommended to reattach it and press the four sites of Slim SIM tightly before trying.

If the Slim SIM cannot be detected?

Please try the original phone number to dial and operate normally. If the original phone number does not work normally, it is recommended to restick it and try again.
If the original phone number cannot be dialed normally, we will provide a free replacement for the Slim SIM.

If Slim SIM card cannot be read after restick?

If the original phone number can be used normally, please inform us the phone model, the Slim SIM may not support this phone model, you can try to replace another phone and try it again.
